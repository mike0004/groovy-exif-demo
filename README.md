# groovy-exif-demo
Sample using groovy with image exif tags using Grape for dependencies

Run this via `groovy GroovyEdifDemo.groovy sample-exif.jpg`  
   or  
 Run it as a script `GroovyEdifDemo.groovy sample-exif.jpg`
